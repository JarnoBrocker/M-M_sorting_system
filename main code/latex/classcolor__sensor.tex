\doxysubsection{Detailed Description}
Color sensor. 

This is a class that will read and write data to the TCS3472 color sensor.~\newline
The i2c\+\_\+bus of hwlib is stored in the class.~\newline
There are a few functions provided like Setup and get\+\_\+data\+\_\+red.~\newline
~\newline
Before you start to read the colors, use the Set\+Up function.~\newline
If Set\+Up returns true, you are ready to go.

Writes the right settings to the color sensor. 

First checks if the ID of the color sensor is 0x44 or 0x4D.~\newline
Another value means that the color sensor isn\textquotesingle{}t connected right.~\newline
After that check, Setup will set the ATIME and the GAIN (both control how precise the sensor is).~\newline
Last step is to turn on the color sensor, so it can \textquotesingle{}see\textquotesingle{} colors.~\newline
